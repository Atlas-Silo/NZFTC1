## Local dev (Blazor Server) - quick start

1. Ensure .NET SDK 8.x is installed.

2. From repo root:
   - Windows (PowerShell):
     
     ./scripts/dev-start.ps1

This runs the server with dotnet watch and serves the Blazor app at:
http://localhost:3000
